# Blue Moor - Learn (iPhone-ready web app)

This is the **shareable iPhone test build**. Open it in **Safari** — no App Store, no developer account.

## Why not a normal app file?

Apple does **not** allow installing a random app package from WhatsApp the way Android allows an APK.  
For friends to try the product today, use this **web app link**.

## Live link (send this on WhatsApp)

**https://pratyush-python.github.io/BlueMoor-Learn/**

## For testers (iPhone)

1. Open the link in **Safari** (not Chrome if possible).
2. Use the app normally (Today / History / Cosmos / Progress).
3. Optional: **Share → Add to Home Screen** for an app icon.

Progress is saved **on that phone’s browser** only.

## Local preview (your Mac)

```bash
cd web
python3 -m http.server 8765
```

Then open `http://localhost:8765` on your Mac, or your Mac’s LAN IP on a phone on the same Wi‑Fi.
